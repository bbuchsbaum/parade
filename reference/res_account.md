# Alias for account()

Alias for account()

## Usage

``` r
res_account(profile, value)
```

## Arguments

- profile:

  A resource profile object

- value:

  Account name

## Value

Updated profile

## Examples

``` r
res_account(profile(), "my-lab-account")
#> Parade Resource Profile
#> Resources:
#>   account: my-lab-account
```
