# Create a tibble/data frame type specification

Create a tibble/data frame type specification

## Usage

``` r
tbl(...)
```

## Arguments

- ...:

  Column specifications for the tibble

## Value

A tibble prototype for schema definitions

## Examples

``` r
schema(data = tbl())
#> # A tibble: 0 × 1
#> # ℹ 1 variable: data <tibble[,0]>
```
