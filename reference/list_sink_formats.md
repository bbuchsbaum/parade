# List registered sink formats

Returns names of all registered sink formats.

## Usage

``` r
list_sink_formats()
```

## Value

Character vector of format names

## Examples

``` r
list_sink_formats()
#> [1] "csv"       "feather"   "json"      "parquet"   "qs"        "rds"      
#> [7] "readr_csv" "readr_tsv" "tsv"      
```
