# Convert profile to list for slurm_resources()

Convert profile to list for slurm_resources()

## Usage

``` r
# S3 method for class 'parade_profile'
as.list(x, ...)
```

## Arguments

- x:

  A resource profile object

- ...:

  Additional arguments (unused)

## Value

List of resources

## Examples

``` r
if (FALSE) { # \dontrun{
prof <- profile_get("standard")
as.list(prof)
} # }
```
